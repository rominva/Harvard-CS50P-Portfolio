# Harvard CS50P Portfolio

A collection of projects and problem sets completed while studying **CS50's Introduction to Programming with Python**.

This repository serves as a record of my progress and learning journey in Python programming.

## Topics Covered

- Python Fundamentals
- Functions and Modules
- Command-Line Arguments
- Exception Handling
- File I/O
- Regular Expressions
- Object-Oriented Programming
- Unit Testing
- APIs and JSON
- Data Validation

## Repository Structure

Each project is stored in its own directory and includes a dedicated README file containing:

- Project description
- Usage instructions
- Concepts practiced
- Implementation notes

## Purpose

The goal of this repository is to document my hands-on practice while learning Python through Harvard University's CS50P course and to showcase my growth as a programmer.
