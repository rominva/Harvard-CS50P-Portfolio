# CS50P – Problem Set 6: File I/O

Solutions to **Problem Set 6 (File I/O)** from **Harvard University's CS50's Introduction to Programming with Python (CS50P)**.

## Overview

This problem set introduces working with files in Python, including reading and writing text and CSV files, processing command-line arguments, handling exceptions, and using external libraries to manipulate images.

## Problems

* 📄 **Lines of Code** – Count the number of lines of Python code while ignoring blank lines and comments.
* 🍕 **Pizza Py** – Read data from a CSV file and display it as a formatted table.
* 🔄 **Scourgify** – Read, clean, and rewrite CSV data into a new format.
* 👕 **CS50 P-Shirt** – Resize an image and overlay it with a shirt image using the Pillow library.

## Topics Practiced

* File input and output
* Context managers (`with`)
* Command-line arguments (`sys.argv`)
* Exception handling
* CSV file processing (`csv`)
* Image processing with Pillow (`PIL`)
* Working with third-party Python packages

## Skills Developed

* Reading and writing files
* Processing structured CSV data
* Validating command-line input
* Handling file-related exceptions
* Manipulating images programmatically
* Writing clean, modular, and maintainable Python code

## Notes
